# Social Hub Django Project

A simple social media web app built with Django. Users can register, log in, create posts, like/favorite posts, send friend requests, and manage their profile with avatar support.

## Features

- User registration and login with custom user model (email as username)
- Profile page with avatar and bio
- Create, edit, delete posts
- Like and favorite posts
- Send and accept friend requests
- Paginated post lists
- Responsive design (manual form styling)
- **Request admin access via email feature** â€” users can send a request to become an admin

## Tech Stack

- Python 3.x
- Django 4.x
- SQLite (default DB)
- HTML, CSS (manual styling)
- JavaScript (for avatar modal)
